# Kedy on Github

Welcome to the official Hugging Face organization for **Kedy** — building next-generation **Small Language Models (SLMs)**.

We focus on creating **efficient, high-performance open-source models** while also delivering **enterprise-grade AI solutions**.

---

## 🚀 Our Models

### 🧠 Kedy-SLM Series

- **Atlas**
  - Lightweight, fast, and optimized for edge and low-latency applications.
  - Designed for developers who need performance without massive compute.

- **Orion**
  - Instruction-tuned model for conversational AI and task execution.
  - Ideal for assistants, automation, and productivity tools.

- **Maxwell**
  - Designed for scientific reasoning, research workflows, and technical domains.
